* Initial release of the PHP language library: AST, control-flow graph (with real branching for
  conditionals, loops, `switch`, `match`, `try`/`catch` and short-circuit operators), SSA, a
  field-sensitive interprocedural data-flow and taint-tracking library on the shared engine, lightweight
  type inference with virtual dispatch, and Models-as-Data extension points for sources, sinks,
  sanitizers and flow summaries.
